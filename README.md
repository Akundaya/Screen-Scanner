# Screen-Scanner
To scan student's screens for online invigilation


Dependencies
1) cv2
2) numpy
3) Test2.mp4 (or any video test)

Steps to use code:
1) Download the main code and its dependencies
2) Place them in the same file folder
3) Run FindScreen.py
4) Done
